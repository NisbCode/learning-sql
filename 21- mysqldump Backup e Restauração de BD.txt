#O QUE VAMOS FAZER#

Faremos o backup de um banco de dados que está rodando no MySQL em um servidor Linux. 
Para realizar backup e restauração rodando o MySQL  no Microsoft Windows, use o PHPMyAdmin.


#COMO FAZER#

Para isso vamos usar o comando mysqldump, que será executado no terminal do Linux, de acordo com a sintaxe a seguir 
(criando o arquivo de backup cujo nome você deverá especificar, com a extensão .sql):

mysqldump -u root -p nome_banco > arquivo_backup.sql


#EXEMPLOS#

Por exemplo, para realizarmos o backup do banco de dados db_Biblioteca no diretório do usuário fabio usamos o comando a seguir

mysqldump -u root -p db_biblioteca > C:\Users\natyb\Desktop\MySQL\Backup\db_biblioteca.sql

#RESTARAÇÃO DO BANCO DE DADOS#

Se o banco de dados apresentar algum problema e precisar ser restaurado a partir do backup, siga o procedimento a seguir:

				1- Crie um banco de dados novo no servidor, com o nome que desejar. Por exemplo, “teste-restore”;

				2- Execute o seguinte comando (no terminal):

mysql -u root -p banco_criado < backup.sql


#EXEMPLO2#

		Crie um novo banco denominado teste-restore; Restaure o banco de dados db_Biblioteca digitando o comando a seguir:

mysql -u root -p teste-restore < /home/fabio/db_Biblioteca.sql
